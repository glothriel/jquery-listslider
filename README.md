# jquery-listslider
Plugin  makes horizontal lists to collapse if they don't have enough space, then shows pagination allowing to slide or swipe
to see hidden elements. 

usage: 


    <ul class="my-listslider">
        <li>Ziomek</li>
        <li>Romek</li>
        <li>Atomek</li>
        <li>Lizus</li>
        <li>Mizus</li>
      `<li>sizus</li>
    </ul>

    <script>
        $('.my-listslider').listslider();
    </script>
